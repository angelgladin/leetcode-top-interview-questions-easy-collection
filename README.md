### How to use [file_creator.py](file_creator.py)?

Example,

```bash
$ python3 file_creator.py -i kt -o ts -f .
```

- `kt`: Stands for _Kotlin_ input file extention.
- `ts`: Stands for _TypeScript_ output file extention.
- `.`: The path of the _folder_.
